admin namespace

admin_addPeer​

Adds given node.

  • Parameters
  • Request
  • Response

  1. enode: string

  2. addToStaticNodes: boolean

curl localhost:8545 \
  -X POST \
  -H "Content-Type: application/json" \
  --data '{
      "jsonrpc": "2.0",
      "id": 0,
      "method": "admin_addPeer",
      "params": [enode, addToStaticNodes]
    }'

Added node

{
  "jsonrpc": "2.0",
  "id": 0,
  "result": result
}

result: string

admin_peers​

Displays a list of connected peers including information about them (clientId, host, port, address, isBootnode, isStatic, enode).

  • Parameters
  • Request
  • Response
  1. includeDetails: boolean
curl localhost:8545 \
  -X POST \
  -H "Content-Type: application/json" \
  --data '{
      "jsonrpc": "2.0",
      "id": 0,
      "method": "admin_peers",
      "params": [includeDetails]
    }'

List of connected peers including information

{
  "jsonrpc": "2.0",
  "id": 0,
  "result": result
}

result: array of object

  • address: string
  • clientType: string
  • enode: string
  • ethDetails: string
  • host: string
  • id: string
  • inbound: boolean
  • isBootnode: boolean
  • isStatic: boolean
  • isTrusted: boolean
  • lastSignal: string
  • name: string
  • port: string (hex integer)
