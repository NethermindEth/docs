personal namespace

personal_importRawKey​

  • Parameters
  • Request
  • Response

  1. keyData: string (hex data)

  2. passphrase: string

curl localhost:8545 \
  -X POST \
  -H "Content-Type: application/json" \
  --data '{
      "jsonrpc": "2.0",
      "id": 0,
      "method": "personal_importRawKey",
      "params": [keyData, passphrase]
    }'
{
  "jsonrpc": "2.0",
  "id": 0,
  "result": result
}

result: string (address)
